Andrew Feenberg resiste ao lado fatalista de Heidegger. Concorda que a tecnologia não é neutra, mas argumenta que as tecnologias são socialmente moldadas e politicamente contestáveis.

## Tese central

> As tecnologias não são destino. Podem ser redesenhadas. A composição (Gestell) não é fado.

O design tecnológico incorpora valores — aquilo a que Feenberg chama **códigos técnicos**. Os sistemas podem ser concebidos para o controlo ou para a participação, para o lucro ou para o bem público, para a vigilância ou para a privacidade, para a eficiência ou para o cuidado, para a hierarquia ou para a democracia.

## Racionalização democrática

A alternativa de Feenberg: cidadãos, utilizadores, trabalhadores e comunidades devem participar no design tecnológico. Isto não é ser anti-tecnologia — é tecnologia *democrática*. Exemplos incluem os movimentos de código aberto, o design participativo, a ciência cidadã e as infraestruturas de controlo comunitário.

## A correção política

Onde Heidegger diagnostica a essência da tecnologia moderna como composição (Gestell), Feenberg responde: sim, mas as tecnologias são múltiplas, contestadas e abertas ao redesenho. A questão não é "o que é a tecnologia?" mas "qual tecnologia, concebida por quem, para quem?"
