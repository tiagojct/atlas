A capacidade de ler, interpretar, criticar e comunicar dados de saúde — um pré-requisito para a prática baseada em evidência no século XXI.

## Por que razão é importante

Os profissionais de saúde são treinados para ler normas de orientação clínica, não para interrogar os dados que as produziram. Quando a IA começar a gerar essas normas — ou a afirmar que as gera — a capacidade de avaliar o pipeline de dados torna-se competência clínica.

## Componentes

- Compreender a proveniência dos dados: de onde veio este número?
- Reconhecer padrões de omissão: quem não está neste conjunto de dados?
- Interpretar a incerteza: qual é o intervalo, não apenas a estimativa pontual?
- Detetar confundimento por indicação em dados observacionais.

## A lacuna educacional

A maioria dos currículos de saúde ensina estatística como um módulo autónomo. A literacia de dados de saúde requer integrar o raciocínio sobre dados em todas as disciplinas clínicas — não como uma ilha separada.
