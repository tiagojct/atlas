Um [[20260524090009|teorema]] auxiliar. É formalmente demonstrado, mas o seu interesse é instrumental: serve para demonstrar outro resultado, mais importante.

Um lema raramente é citado por si mesmo; é uma peça de ligação dentro de uma demonstração maior. A palavra vem do grego *lêmma*, «algo que se recebe ou toma» — o lema toma-se como um passo no caminho para o teorema principal.

## Exemplo

O **Lema de Zorn** é um resultado importante da teoria dos conjuntos, mas o seu nome como «lema» reflete o papel que originalmente desempenhou: Zorn provou-o para o usar na demonstração de outro resultado.
