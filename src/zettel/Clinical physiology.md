The discipline that bridges basic physiology and clinical medicine — measuring function to inform diagnosis, prognosis, and treatment.

Clinical physiologists do not simply run tests; they interpret physiological signals in context. A spirometry curve is not a number — it is a story about airways, effort, and disease progression.

## Core domains

- Respiratory physiology (spirometry, lung volumes, diffusing capacity, FeNO).
- Cardiovascular physiology (ECG, exercise testing, Holter).
- Neurophysiology (EEG, evoked potentials, sleep studies).
- Exercise physiology (cardiopulmonary exercise testing).

## The mindset

The clinical physiologist's question is not "is this value normal?" but "does this pattern of values make physiological sense in *this* patient?". Pattern recognition plus mechanistic reasoning.
