O Quartz é um gerador de sites estáticos open-source concebido para publicar vaults do Obsidian como jardins digitais. Criado por Jacky Zhao.

## O que faz

O Quartz recebe uma pasta de ficheiros Markdown (um diretório de conteúdo de um vault Obsidian), resolve `[[wikilinks]]`, constrói uma vista de grafo e gera um site HTML estático — sem servidor, sem base de dados.

## Funcionalidades principais

- **Resolução de wikilinks:** `[[note-id]]` torna-se um link real.
- **Backlinks:** cada página mostra o que aponta para ela.
- **Vista de grafo:** visualização espacial da rede de notas.
- **Pesquisa de texto integral:** via Pagefind.
- **Temas personalizados:** baseados em CSS, compatíveis com Pequod.

## O pipeline de publicação

Obsidian (escrever) → Quartz (compilar) → GitHub Pages ou Cloudflare (servir). As alterações são submetidas ao git, compiladas em CI, implementadas automaticamente.
