Obsidian is a local-first, plain-text Markdown editor with a plugin ecosystem that makes it a powerful academic tool — not just a note-taking app.

## The core workflow

1. **Zettelkasten:** atomic notes with `[[wikilinks]]` in a flat or minimally structured folder.
2. **Daily notes:** timestamped logs of meetings, ideas, decisions.
3. **Reference management:** Zotero integration via plugins (Citations, Zotero Integration).
4. **Long-form writing:** essays and papers live in a `Writings/` folder, linked to Zettelkasten notes for reflection.
5. **Publishing:** static-site generators turn the vault into a public site (this Atlas is built that way).

## Why plain text matters

Markdown files are future-proof. They can be read by any editor, versioned with git, and processed by any tool. No proprietary lock-in.
