Writing changed human memory. Photography changed it again. Digital storage changes it further still — turning recollection from an inner act into an external archive.

## The externalisation of memory

Memory is increasingly stored in photo libraries, cloud storage, search histories, chats, calendar reminders, social timelines, and recommendation feeds. This grants extraordinary powers of preservation — but it transforms what remembering is.

## The subtle loss

A remembered event can become less something *inwardly digested* and more something *retrieved as data*. We may photograph an experience before fully living it. We may recall through the archive rather than through narrative.

## Memory is not storage

Human memory is not just storage. It is interpretation. It involves forgetting, reshaping, mourning, forgiving, and making meaning. A total archive may preserve more — but not necessarily help us understand more. Forgetting is not always a failure of memory; sometimes it is a condition of healing.

## Stiegler's frame

[[Bernard Stiegler — technics and time|Stiegler's]] concept of [[Tertiary retention|tertiary retention]] — memory stored in technical objects — provides the richest vocabulary for understanding how external memory supports reorganise human time, desire, and culture.
